## Consultation UI mockup (export to JPG)

### Files
- `design/consultation-ui-mockup.svg`: static mockup of the consultation screen (video + AI questions + end CTA + transcript).

### Export to JPG
- **macOS (Preview)**: open the SVG → `File` → `Export...` → choose `JPEG`.
- **Chrome**: open the SVG → zoom to fit → take a screenshot (or print to PDF then convert).
